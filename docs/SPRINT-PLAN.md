## 📦 Phase 1: 数值系统重做（玩法核心）

> 当前数值体系和原版差距较大，这是影响"玩起来像不像"的关键。

### 1.1 经济系统对齐原版
**当前问题**：
- 初始金 100、木 0，原版是 ~800 金 + 5-10 木
- 杀怪奖金太低（2-10），不够建塔节奏
- 木材只能金换木，原版每波自动奖励木材
- 人口上限 20 固定，原版可以用木材换人口

**修改内容**：
- [ ] 初始金 800，木 5
- [ ] 每波通关奖励：金 50 + 波数×5，木 1-2
- [ ] 杀怪奖金按怪物类型调整（普通 1-3，精英 5-10，Boss 50-200）
- [ ] 人口上限初始 5，每 12 木换 +1 人口（和现有一致但初始降低）
- [ ] 每波利息：当前金×2%（鼓励存钱策略）

**验收**：前 10 波能建 3-5 座塔 + 英雄，不太富也不太穷

### 1.2 英雄系统对齐原版
**当前问题**：
- 英雄不占人口（原版不同强度英雄占不同人口）
- 技能学习只需技能点（原版需要金钱/木材）
- 经验获取太快（杀一个怪就加大量经验）
- 属性点免费加（原版部分需要金钱）
- 英雄太弱，carry 不了

**修改内容**：
- [ ] 英雄按强度占不同人口：
  - easy 难度英雄（神牛、寒冰、战神）：占 2 人口
  - medium 难度英雄（火魂、剑神、电法、恶魔之子）：占 3 人口
  - hard 难度英雄（小黑、水魔、风神、兽王、幻影）：占 4 人口
  - HeroTowerConfig 新增 `populationCost` 字段
- [ ] 技能学习消耗：1 级免费，2 级 100 金，3 级 300 金 + 1 木，4 级 800 金 + 3 木，5 级 2000 金 + 5 木
- [ ] 经验曲线调缓：升级所需经验 = 等级² × 8（1→8, 2→32, 10→800）
- [ ] 杀怪经验 = 怪物 goldReward × 1.5（精英/Boss 给更多）
- [ ] 属性点加点保持免费（简化）
- [ ] 英雄伤害公式加强：baseDamage + str×1.5（现在是 str×0.8）
- [ ] 英雄攻速公式：max(500, base - agi×3)（让敏捷更有用）
- [ ] 英雄射程：baseRange + int×0.5（智力加成翻倍）
- [ ] **召唤技能真实实现**（当前是假的，只加了点伤害）：
  - 召唤物是独立的"小塔"，放在英雄身旁空格，自动攻击
  - 召唤物有独立的 HP、伤害、攻速、存活时间
  - 技能等级影响召唤物数量和强度
  - 召唤物占用临时人口（消失后归还）
  - 需要实现的英雄：
    - **水魔**：召唤水人（远程魔法攻击，带减速）
    - **兽王**：召唤战鹰（远程穿刺）+ 召唤巨熊（近战高伤害）
    - **恶魔之子**：召唤小鬼（远程混沌攻击）
  - 召唤物需要 3D 模型（简单低多边形即可）
  - 召唤物死亡/到期有消散特效

**验收**：一个中等水平的英雄（20 级左右）+ 5-6 座辅助塔能打到 30 波

### 1.3 塔造价 / 升级成本重新平衡
**当前问题**：
- 多重塔 80 金就能买到 AOE，太便宜
- 高级塔（秒杀塔 40000）几乎买不起
- 升级成本曲线不平滑

**修改内容**：
- [ ] 基础塔：80-150 金（保持）
- [ ] 中级塔：200-500 金
- [ ] 高级塔：800-2000 金
- [ ] 顶级塔：3000-8000 金（从 40000 降下来）
- [ ] 升级成本 = 建造成本 × (0.8 + 等级×0.4)
- [ ] 部分高级塔需要木材（如秒杀塔 5000 金 + 10 木）

**验收**：50 波通关时总花费约 30000-50000 金 + 30-50 木

## 📦 Phase 2: 视觉打磨（美术升级）

> 从"程序员方块"到"魔兽风格"

### 2.1 地形材质升级
**当前问题**：草地/路面都是纯色 MeshLambertMaterial，像塑料

**修改内容**：
- [ ] 草地：用 canvas 生成噪点纹理 → MeshStandardMaterial（带 roughness/normalMap）
- [ ] 路面：石砖纹理（canvas 画砖块图案）+ 法线凹凸感
- [ ] 泥土岸壁：加纵向条纹纹理模拟泥土层理
- [ ] 整体地形加微幅 displacement（起伏更自然）

**验收**：截图对比，草地有纹理层次感，路面有石砖感

### 2.2 水面 / 环境氛围
**当前问题**：内圈建造区是空白的天蓝色（地面穿帮看到背景色）

**修改内容**：
- [ ] 内圈铺一层草地地面（和外圈一样）
- [ ] 地图四周加淡色边缘过渡（不要硬切到天空色）
- [ ] 加环境粒子：缓慢飘动的萤火虫/花瓣（10-20 个）

**验收**：内圈不再是空白色，地图边缘过渡自然

### 2.3 塔模型精细化
**当前问题**：塔模型虽然有区分但多边形数太低，细节不够

**修改内容**：
- [ ] 所有塔底座统一改为六边形（更魔兽风格）
- [ ] 顶部武器部件加细节（箭塔加弓弦、炮塔加火焰口、冰塔加冰晶碎片）
- [ ] 塔身加装饰条纹（用线段画纹路）
- [ ] 光环塔的浮空效果加上下浮动动画

**验收**：截图对比，塔有辨识度且不像方块堆

### 2.4 怪物模型精细化
**当前问题**：小怪物太小看不清，Boss 只是大方块

**修改内容**：
- [ ] 小怪物加更明显的头/手臂（不只是方块）
- [ ] Boss 加发光效果（emissive 脉冲）
- [ ] 飞行怪加翅膀扇动动画
- [ ] 怪物死亡效果加强（更多粒子、更大爆炸）

**验收**：能一眼区分怪物类型

### 2.5 UI 美化
**当前问题**：商店/信息面板是纯色方块，像调试界面

**修改内容**：
- [ ] 商店面板加深色半透明背景 + 金色边框
- [ ] 塔图标用 canvas 画小预览图（不是纯色方块）
- [ ] 顶栏资源图标用 emoji/图形替代纯文字
- [ ] 信息面板加渐变背景 + 圆角

**验收**：UI 看起来像游戏界面而不是 debug 工具

## 📦 Phase 3: 游戏体验（锦上添花）

### 3.1 新手引导
- [ ] 第一次进入游戏时，前 3 波显示操作提示（"点击商店选塔"→"点击地图建造"→"按 N 提前下一波"）
- [ ] 英雄选择界面加推荐标记（难度=easy 的标绿色推荐）

### 3.2 统计面板增强
- [ ] 游戏中按 Tab 查看实时统计：DPS 排行（每座塔的输出占比）
- [ ] 游戏结束时显示详细统计（总伤害、击杀最多的塔、英雄最终等级）

### 3.3 音效完善
- [ ] 英雄升级音效（区别于普通塔升级）
- [ ] 英雄技能释放音效
- [ ] 环境音效（风声/鸟叫，低音量循环）
