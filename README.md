# 绿色循环圈 Web 版 (Green Circle TD)

基于魔兽争霸3经典塔防地图「绿色循环圈」原版 v9.73 的 Web 复刻版。

## 🎮 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`

## 🎯 游戏规则

- 怪物在**闭合环形跑道**上无限循环奔跑
- 在跑道两侧建造防御塔消灭怪物
- **失败条件**: 场上怪物 > 100 或 BOSS 限时未击杀
- **胜利条件**: 消灭全部 50 波 → 隐藏关(10波) → 无尽模式
- 每波在下一波前清空怪物可获得 **PF 点数**，40PF 解锁无尽模式
- 开局随机获得 1 个**英雄成长塔**，击杀获得经验升级

## ⌨️ 操作

| 操作 | 说明 |
|---|---|
| 鼠标点击商店 | 选择塔 |
| 鼠标点击地图 | 放置塔 / 选中已有塔 |
| Shift + 点击 | 连续建造同种塔 |
| 1-9, 0 | 快捷选择塔 |
| U | 升级选中的塔 |
| S | 出售选中的塔 |
| N | 立即开始下一波 |
| P | 暂停/继续 |
| Space | 切换速度 x1/x2/x3 |
| H | 攻击/护甲克制表 |
| M | 移动英雄塔位置 |
| ESC | 取消选择 |

## ⚔️ 攻击/护甲克制

7 种攻击类型 × 8 种护甲类型的完整克制矩阵（基于魔兽争霸3原版数据）。游戏内按 **H** 键查看。

护甲类型：无甲 / 轻甲 / 中甲 / 重甲 / 加强甲 / 英雄甲 / 神圣甲 / 普通甲

怪物特性：🦅飞行 / 👁隐形 / 🛡魔免 / ☠毒免

## 💰 资源系统

- **金钱**: 击杀怪物获得，用于建造/升级
- **木材**: 5000金 → 10木
- **人口**: 12木 → +1人口上限，限制可建造塔数量
- **回收**: 第1波100%，之后75%

## 🛠️ 技术栈

- **Three.js** — 3D 渲染引擎（纯 Three.js，无 Phaser）
- **TypeScript** — 类型安全
- **Vite** — 快速构建
- **Web Audio API** — 合成音效（零外部资源依赖）
- **HTML/CSS** — UI 层（商店/信息面板/顶栏/结算）

## 📁 项目结构

```
src/
├── main.ts                # 入口
├── config/
│   ├── towers.ts          # 16种塔配置
│   ├── enemies.ts         # 40种怪物配置
│   ├── waves.ts           # 50波配置
│   └── heroTowers.ts      # 12种英雄塔配置
├── entities/
│   ├── Tower.ts           # 防御塔（暴击/弹射/秒杀/光环）
│   ├── Enemy.ts           # 怪物（飞行/隐形/魔免/毒免/降甲）
│   ├── HeroTower.ts       # 英雄成长塔（属性/技能/经验）
│   └── Projectile.ts      # 弹道
├── systems/
│   ├── PathManager.ts     # 闭合跑道路径
│   ├── GridManager.ts     # 网格建造
│   ├── EconomyManager.ts  # 金钱/木材/人口
│   ├── WaveManager.ts     # 波次+隐藏关+无尽
│   └── SoundManager.ts    # Web Audio 合成音效
├── scenes/
│   ├── BootScene.ts
│   ├── MainMenuScene.ts   # 难度选择
│   ├── GameScene.ts       # 核心游戏
│   └── GameOverScene.ts   # 评级结算
└── utils/
    ├── constants.ts       # 常量+7×8克制矩阵
    └── helpers.ts         # 伤害计算
```
