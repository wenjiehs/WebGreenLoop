# 绿色循环圈 Web 复刻版 - 设计文档

## 概述
基于魔兽争霸3经典塔防地图"绿色循环圈"(Green Circle TD) 原版 v9.8-v10.1 的 Web 复刻版。

## 技术栈
- **引擎**: Phaser 3 + TypeScript
- **构建**: Vite
- **美术**: 魔兽风格（分阶段替换）

## 核心机制
- 闭合环形跑道，怪物无限循环
- 失败条件：怪物 > 100 或 BOSS 超时
- 10种塔 + 攻击/护甲克制矩阵
- 50+波 + BOSS + 隐藏关 + 无尽模式
- PF点数系统（40PF开启无尽）
- 金钱/木材/人口经济系统

## 分阶段计划
1. Phase 1: 核心骨架（地图+跑道+怪物循环+基础塔+金钱）
2. Phase 2: 完整塔防（10种塔+升级+克制+5种怪物）
3. Phase 3: 波次系统（50波+BOSS+PF+难度）
4. Phase 4: 体验打磨（UI+音效+美术+隐藏关/无尽）
